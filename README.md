## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Performed an initial data inspection.

### 2. Exploratory Data Analysis (EDA)

* Checked data types and summary statistics.
* Identified missing values.
* Explored patterns and trends using visualizations.

### 3. Data Cleaning

* Removed duplicates.
* Handled missing values.
* Corrected data formats.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported cleaned data into PostgreSQL/MySQL/SQL Server.
* Executed SQL queries to answer business questions.
* Used filtering, grouping, joins, aggregations, and sorting.

### 5. Dashboard Development

* Built an interactive Power BI dashboard.
* Added KPIs, charts, slicers, and filters.
* Visualized key business insights.

### 6. Reporting

* Created a project report summarizing findings.
* Designed a presentation using Gamma AI.

2. Install the required Python libraries.

6. Execute the SQL queries.
7. Open the Power BI file to explore the dashboard.
